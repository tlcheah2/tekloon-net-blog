## Problem

We could only select/upload one image file at a time with below code.

```tsx
<input
    className="hidden"
    type="file"
    accept="image/*"
    onChange={handleImageUpload}
    ref={imagesInputRef}
/>
```

## Solution

We could enable multiple file selection by adding `multiple` attribute to the input element.

```tsx
<input
    multiple
    className="hidden"
    type="file"
    accept="image/*"
    onChange={handleImageUpload}
    ref={imagesInputRef}
/>
```
